1.¿QUE ES UNA RED?
UNA CONEXION DE VARIOS DISPOSITIVOS, MEDIANTE LA CUAL SE TRANSFIEREN DATOS
2.¿QUE ES EL MODELO TCP/IP?
ES UN PROTOCOLO EN EL CUAL PODEMOS CONOCER COMO VIAJA LA INFORMACION
3.¿QUE ES UN IP?
ES LA DIRECCION QUE IDENTIFICA LA CONEXION QUE SE TIENE A LA RED
4.¿QUE ES EL PROTOCOLO TCP?
ES EL PROTOCOLO QUE DEBE DE SEGUIRSE PARA QUE EXISTA UN INTERCAMBIO DE DATOS DE LA MANERA MAS OPTIMA Y SEGURA
5.DENTRO DEL PROTOCOLO TCP,¿QUE ES EL PROTOCOLO HTTP?
PROTOCOLO QUE SE REALIZA PARA QUE LA INFORMACION VIAJE DE MANERA SEGURA EN UNA RED GLOBAL
6.PUERTO DEL PROTOCO HTTPS
PUERTO EN EL QUE SE CIFRA LA INFORMACION PARA QUE VIAJE DE MANERA SEGURA EN UNA RED GLOBAL 
7.DIFERENCIA ENTRE HTTP Y HTTPS
EN HTTP LA INFORMACION VIAJA DE MANERA PLANA, Y EN HTTPS LA INFORMACION SE CIFRA PARA QUE EL TRANSPORTE SEA MAS SEGURO
8.¿QUE SIGNIFICA EL MODELO CLIENTE - SERVIDOR?
LA INTERACCION QUE EXISTE ENTRE UNO Y OTRO SIGUIENDO CIERTAS NORMAS PARA QUE SE LLEVE A CABO LA CONEXION MAS SEGURA
9.¿QUE ES UNA PETICION (REQUEST)?
PETICION MEDIANTE LA CUAL SE SOLICITA CIERTA INFORMACION AL SERVIDOR
10.¿QUE ES UNA PETICION (RESPONSE)?
ES LA SOLICITUD DE RESPUESTA DEL CLIENTE AL SERVIDOR
11.SECCIONES DE UNA PETICION
GET, POST, PUT
